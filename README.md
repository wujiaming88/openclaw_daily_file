# 多源科技新闻摘要生成工具

这个工具可以从多个国际新闻源抓取科技新闻，并使用BytePlus API生成综合摘要。

## 功能特性

- 📰 支持10个主流国际新闻源
- 🤖 使用BytePlus byteplus/seed-1-8-251228模型生成智能摘要
- 📊 突出不同来源的观点差异
- 💾 自动保存摘要到文件
- 🔄 兼容系统环境变量，无需额外配置API密钥

## 支持的新闻源

1. BBC
2. CNN
3. 路透社
4. 纽约时报
5. 华尔街日报
6. 金融时报
7. 彭博社
8. 法新社
9. 美联社
10. 每日邮报

## 安装依赖

```bash
npm install
```

## 配置环境变量

### 自动配置（推荐）
工具会自动使用系统已配置的BYTEPLUS_API_KEY或OPENAI_API_KEY环境变量，无需额外配置。

### 手动配置
复制`.env.example`为`.env`并填入你的BytePlus API密钥：

```bash
cp .env.example .env
```

编辑`.env`文件：

```env
BYTEPLUS_API_KEY=your_byteplus_api_key_here
```

## 使用方法

```bash
npm start
```

## 注意事项

1. 确保你的BytePlus API密钥有足够的余额
2. 部分新闻源可能有反爬虫机制，工具会自动处理基本的反爬策略
3. 建议不要过于频繁地运行，避免给新闻源服务器造成压力
4. 生成的摘要可能存在延迟，取决于网络状况和BytePlus API响应速度
5. 工具会优先使用BYTEPLUS_API_KEY环境变量，若不存在则 fallback 到OPENAI_API_KEY
