# OpenClaw Daily Files

这个仓库用于存储OpenClaw系统自动生成的日报文件，包括：

## 目录结构

### daily_news/
- 每日新闻联播整理
- 格式：YYYY-MM-DD.md
- 内容：央视新闻联播主要内容摘要

### daily_ai_news/
- 每日AI科技新闻整理
- 格式：YYYY-MM-DD.md
- 内容：中美主要科技公司的最新动态和技术进展

### weekly_github_trending/
- 每周GitHub趋势报告
- 格式：YYYY-WW.md
- 内容：GitHub上本周最受欢迎的开源项目

### system_health/
- 系统健康检查报告
- 格式：YYYY-MM-DD.md
- 内容：系统资源使用情况、性能指标和健康分析

## 更新频率

- daily_news: 每天09:00（北京时间）
- daily_ai_news: 每天15:00（北京时间）
- weekly_github_trending: 每周一03:00（北京时间）
- system_health: 每天12:00（北京时间）

## 使用说明

所有文件均由OpenClaw系统自动生成和推送，请勿手动修改。如需调整更新频率或内容范围，请联系系统管理员。
